# 字体文件说明

## 字体目录

此目录用于存放表情包生成器使用的字体文件。

## 支持的字体格式

- TrueType字体 (.ttf)
- OpenType字体 (.otf)
- TrueType集合 (.ttc)

## 字体获取建议

### 免费中文字体（可自由使用）
1. **思源黑体** (Source Han Sans) - Google和Adobe开发
2. **文泉驿微米黑** - 开源中文字体
3. **方正黑体**（需确认授权）
4. **站酷系列字体** - 部分免费商用

### 英文字体
1. **Arial** - Windows系统自带
2. **Helvetica** - macOS系统自带
3. **Roboto** - Google开源字体
4. **Open Sans** - 开源字体

### 创意字体
1. **Impact** - 经典表情包字体
2. **Comic Sans MS** - 手写风格
3. **Bebas Neue** - 无衬线标题字体

## 字体安装方法

1. **方法一：系统字体**
   - 将字体文件复制到系统字体目录
   - Windows: `C:\Windows\Fonts\`
   - macOS: `/Library/Fonts/` 或 `~/Library/Fonts/`
   - Linux: `/usr/share/fonts/` 或 `~/.fonts/`

2. **方法二：项目字体**
   - 将字体文件放入此目录 (`assets/fonts/`)
   - 程序会自动加载此目录下的所有字体文件
   - 支持子目录结构

## 配置优先级

字体加载优先级（从高到低）：
1. 用户自定义字体（通过配置指定）
2. 项目字体目录 (`assets/fonts/`)
3. 系统字体
4. 默认回退字体

## 最佳实践

### 1. 字体选择
- **中文场景**：使用中文字体，确保字符集完整
- **英文场景**：使用英文字体，渲染效果更好
- **混合场景**：使用支持多语言的字体（如思源黑体）

### 2. 字体大小
- **标题文字**：24-36像素
- **正文文字**：18-24像素
- **小字注释**：12-16像素

### 3. 颜色对比
- 浅色背景使用深色文字
- 深色背景使用浅色文字
- 添加文字描边增强可读性

### 4. 性能考虑
- 避免使用过多字体文件
- 字体文件大小适中
- 考虑使用字体子集

## 常见问题

### 1. 字体不显示
- 检查字体文件路径是否正确
- 确认字体格式是否支持
- 查看程序日志获取错误信息

### 2. 文字渲染问题
- 中文字符显示为方框：缺少中文字体
- 文字位置偏移：字体度量信息不准确
- 文字模糊：字体大小或抗锯齿设置

### 3. 版权问题
- 确保字体有商用授权
- 使用开源或免费商用字体
- 尊重字体设计师的劳动成果

## 字体配置文件示例

```json
{
  "default_fonts": {
    "chinese": "SourceHanSansCN-Regular.ttf",
    "english": "Roboto-Regular.ttf",
    "title": "Impact.ttf"
  },
  "font_sizes": {
    "small": 16,
    "medium": 24,
    "large": 36
  },
  "font_colors": {
    "light_bg": "#000000",
    "dark_bg": "#FFFFFF"
  }
}
```
