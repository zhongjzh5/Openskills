# 背景图片资源

## 目录说明

此目录用于存放表情包生成的背景图片资源。

## 背景图片用途

1. **模板背景**：作为表情包的基础背景
2. **文字底图**：文字叠加的背景图片
3. **合成素材**：多图合成的素材图片
4. **装饰元素**：边框、图标等装饰性图片

## 图片要求

### 格式要求
- **推荐格式**：PNG（支持透明通道）
- **备用格式**：JPG、JPEG
- **尺寸要求**：建议 600×400 像素以上
- **分辨率**：72-150 DPI

### 内容要求
1. **简洁清晰**：避免过于复杂的背景干扰文字
2. **适当留白**：为文字预留足够空间
3. **颜色对比**：确保文字在背景上清晰可读
4. **版权合规**：使用无版权或已获授权的图片

## 背景分类

### 按内容分类
- **纯色背景**：单一颜色，简洁明了
- **渐变背景**：颜色渐变，视觉效果丰富
- **纹理背景**：添加纹理质感
- **场景背景**：具体场景（办公室、家庭、户外等）
- **抽象背景**：抽象图案或艺术效果

### 按用途分类
- **通用背景**：适合多种场景
- **节日背景**：特定节日主题
- **话题背景**：特定话题相关（工作、学习、娱乐等）
- **情感背景**：表达特定情感（开心、悲伤、愤怒等）

## 使用方法


## 常见问题

### 1. 背景图片不显示
- 检查文件路径是否正确
- 确认文件格式是否支持
- 验证文件权限

### 2. 文字可读性差
- 调整文字颜色
- 添加文字阴影或描边
- 使用半透明背景框

### 3. 图片质量差
- 使用更高分辨率的图片
- 优化图片压缩参数
- 选择合适的文件格式

### 4. 版权问题
- 确认图片授权信息
- 使用CC0或公共领域图片
- 遵守署名要求
